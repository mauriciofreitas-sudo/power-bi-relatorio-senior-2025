## Descrição Resumida dos Relatórios

### Financeiro / Contabilidade

- **[Crédito Conta Contábil](docs/credito-conta-contabil.md)** (180 MB) — créditos lançados por conta contábil no Senior. Análise contábil e auditoria de lançamentos.

- **[Dashboard - Mapeamento_Cargas](docs/dashboard-mapeamento-cargas.md)** (1,85 MB) — mapa de rotas, volumes por origem/destino e on-time delivery.

## Notas Operacionais

- Todos os arquivos estão no formato `.pbix` (Power BI Desktop). Para abrir, é necessário o **Power BI Desktop** instalado.
- Arquivos com dados importados (Import Mode) têm os dados embutidos no `.pbix`, por isso o tamanho elevado nos relatórios maiores.
- Para publicar no **Power BI Service**, acessar: Página Inicial > Publicar > selecionar o workspace de destino.
- Ao atualizar dados após publicação, configurar o **gateway de dados** apontando para o banco Senior.
- O arquivo `ChamadosGeral Painel (1).pbix` possui `(1)` no nome — verificar se é duplicata ou versão atualizada do `chamados.pbix`.
- O relatório `Auditoria_Michelin - TV CGR.pbix` é uma versão de TV — não renomear para não perder o contexto de exibição.
