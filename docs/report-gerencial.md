## Objetivo

Relatório gerencial com visão executiva dos principais indicadores do Grupo LC Log. Complementa o `Dashboard - Performance Geral.pbix` com foco em indicadores financeiros e operacionais para apresentação em reuniões de gestão (gerentes e coordenadores).

## Fontes de Dados

- Sistema Senior ERP — múltiplos módulos (financeiro, transportes, frota)
- Dados consolidados por filial / regional

## Principais Indicadores

- Faturamento do período (realizado x meta)
- Custo operacional e margem
- Volume de operações (viagens, entregas, CT-e)
- Inadimplência e posição de caixa
- Indicadores de frota (utilização, CPKm)
- Headcount (se incluído)
- Resultado por filial

## Páginas Esperadas

1. **Resumo Executivo** — KPIs principais com variação vs. período anterior
2. **Financeiro** — receita, custos e resultado
3. **Operacional** — volume e performance
4. **Por Filial** — comparativo entre unidades
5. **Metas** — realizado vs. metas estabelecidas

## Filtros Esperados

- Período (mês/ano)
- Filial / regional
- Área de negócio (se multi-divisão)

## Frequência de Atualização

Mensal — alinhado ao fechamento gerencial.

## Público-Alvo

Gerentes de área, Coordenadores, Diretoria (nível intermediário ao executivo).

## Observações

- Arquivo médio (25 MB) — tamanho adequado para um relatório gerencial.
- Diferença em relação ao `Dashboard - Performance Geral.pbix` (277 MB): este é mais enxuto e voltado para apresentações, enquanto o outro é mais analítico.
- Verificar se os dados são extraídos das mesmas fontes que o relatório de Performance para garantir consistência nos números.
- Avaliar uso de **bookmarks** para facilitar navegação em apresentações.
- Confirmar se inclui comparativo com mesmo período do ano anterior (YoY) — relevante para análise gerencial.
